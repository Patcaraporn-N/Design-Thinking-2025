# [วิเคราะห์ภาพยนต์ยอดนิยม]

Streamlit นี้พัฒนาเพื่อแสดงข้อมูลภาพยนตร์ยอดนิยม เช่น ชื่อเรื่องและคะแนนรีวิว ในหน้าเดียวให้ดูง่าย
กลุ่มเป้าหมายคือผู้ที่สนใจดูภาพยนตร์หรือหาภาพยนตร์ที่น่าสนใจและต้องการใช้คะแนนรีวิวช่วยตัดสินใจ
แอปนี้ช่วยแก้ปัญหาการต้องค้นหาข้อมูลจากหลายแหล่งหลายที่ ทำให้เลือกดูหนังได้สะดวกและรวดเร็วมากขึ้น

## 🚀 ฟีเจอร์หลัก (Features)
- ฟีเจอร์ที่ 1 ดึงข้อมูลรายชื่อซีรีส์จากหน้าเว็บ Rotten Tomatoes อัตโนมัติ
- ฟีเจอร์ที่ 2 แสดงชื่อเรื่องของแต่ละรายการที่พบในหน้าเว็บ
- ฟีเจอร์ที่ 3 ดึงและแสดงคะแนนรีวิวจากนักวิจารณ์ (ถ้าไม่มีจะแจ้งว่า “ไม่มีคะแนน”)
- ฟีเจอร์ที่ 4 แสดงจำนวนเรื่องทั้งหมดที่ดึงมาได้ในหน้านั้น

การสะท้อนคิด (Reflection)
จากการทำโปรเจกต์นี้ ได้เรียนรู้กระบวนการทำงานตั้งแต่การสกัดข้อมูลด้วย Selenium การจัดเก็บข้อมูลเป็นไฟล์ CSV การทำความสะอาดข้อมูลด้วย pandas ไปจนถึงการพัฒนาเว็บแอปด้วย Streamlit และเชื่อมต่อ OMDb API ทำให้เข้าใจภาพรวมของการทำงานแบบครบขั้นตอน (data pipeline) มากขึ้น ไม่ใช่แค่เขียนโค้ดส่วนใดส่วนหนึ่ง แต่ต้องเชื่อมโยงหลายส่วนเข้าด้วยกันให้ทำงานได้จริง
ได้พัฒนาทักษะการอ่านโครงสร้างหน้าเว็บ (DOM) และการเลือก CSS Selector อย่างถูกต้อง รวมถึงเข้าใจปัญหาที่อาจเกิดขึ้น เช่น การโหลดข้อมูลแบบ JavaScript, การเรียก API ซ้ำ, และ error จาก path ไฟล์ ซึ่งทำให้เห็นความสำคัญของการแก้ปัญหาอย่างเป็นระบบ
สิ่งที่ควรปรับปรุงคือการเขียนโค้ดให้มีโครงสร้างที่ชัดเจนมากขึ้น เช่น การแยกฟังก์ชันให้เป็นระเบียบ การใช้ WebDriverWait แทน time.sleep และการจัดการ error ให้ละเอียดกว่านี้ รวมถึงการวางแผนก่อนลงมือทำเพื่อลดการแก้ไขซ้ำหลายรอบ
โดยรวมแล้ว โปรเจกต์นี้ช่วยให้เข้าใจทั้งเชิงเทคนิคและการทำงานจริงมากขึ้น และทำให้สามารถอธิบายเหตุผลของโค้ดแต่ละส่วนได้ชัดเจนกว่าเดิม
